Portfolio

JVM Status Checker

When deploying from Staging to Production, we track the status of our rolling JVM restart with a page that looks like 90% gobbledegook to the average user. I took the initiative to create this little JS tool which, when run in the browser console, parses out just the relevant information and displays it for optimal legibility.


		

		
Transactional Email Populator

                
In order to add design and promotional elements to transactional emails, we had to translate data from XML generated by our OMS into the proprietary format consumable by our email provider. The following Java API parses the XML and, if the customer is contactable, transmits the info to the marketing platform, logging the filename and disposition into an MS SQL Server database. A scheduled batch file moves new XML files into a workspace folder, runs the API, and archives the files, processing thousands of emails per hour. The revamped emails were estimated to have generated over $48K of additional revenue in the first month.

                

		      

		

		
Products Missing From Employee Site

                
Our employee discount is based on products being placed in certain directories, so after each web launch, we need to make sure all relevant items are in the appropriate places. This simple Oracle SQL query finds all current products that are not in any of the employee categories.
